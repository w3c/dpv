Proposal from Georg to map ISO/IEC 42005 AI System Impact Assessment

agreed this is in scope with the following discussion: If a standard provides useful information, then we add / refine concepts in DPV and extensions with a reference to that standard as the source. We only create an extention when the standard must be specifically represented, such as for certification, or when we are modelling preliminary concepts such as those for IEEE P7012 to support the standard.

Proposal from Beatriz to map EU Unfair Consumer Directive

<ghurlbot> Issue 392 Extension for the Unfair Contract Terms Directive (by besteves4)
